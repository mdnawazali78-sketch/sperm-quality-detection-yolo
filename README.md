# AI Driven Sperm Quality Detection

This project uses a YOLO-based deep learning model to detect sperm components (head, neck, tail) from microscopic images.

## Project Overview
The objective of this project is to automate sperm quality detection using computer vision techniques.

## Technologies Used
- Python
- YOLO
- Computer Vision
- NumPy
- Pandas
- Streamlit

## Workflow
1. Dataset annotation
2. Data preprocessing
3. Data augmentation
4. Model training
5. Model evaluation using mAP
6. Deployment using Streamlit

## Application
This system can help assist in automated sperm quality analysis in medical and fertility research.
